# ActiveSniper

ActiveSniper launch callback when find change of specified column in activerecord.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'activesniper'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install activesniper
